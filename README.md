# Deestar Menu Website

This is a *food/recipe menu website* called "Deestar". It showcases different menu items with images, descriptions, and an *Order* button. It also includes a *Contact Form* for users to submit their details.

## Features
- Homepage with title and navigation (Home | Recipe | Contact)
- Menu section with images, titles, descriptions, and order buttons
- Contact form with Name, Age, Email, and Send button
- Responsive layout using CSS (flexbox)
- Clean and modern color scheme for a food website

## Technologies Used
- HTML
- CSS
